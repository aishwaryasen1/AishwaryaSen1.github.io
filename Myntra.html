Fashion Trend Detection and Recommendation Model

Team Size : 4

Project Guide : Self

Project Duration : Sep 2020 - Nov 2020

Motivation

With the rise of fast fashion, it is difficult to keep track of ever-changing trends. A model that would help the user filter the products based on the latest trends will make the shopping experience a lot smoother. Thus, our model detects the latest trends in the market to help the user make the right choice. It would serve as each individual’s own personal, low budget stylist.

Summary

An Instagram-Scraper model was used to collect images of the top fashion influencers after ranking the influencers using Instagram’s algorithms and features such as likes and comments to determine the relevance of the images. This allows us to perceive the most popular fashion trends. Semantic Segmentation using the Mask-RCNN model (pre trained using COCO weights) and Attribute Recognition using an Encoder-Decoder model, the encoder being an InceptionV3 model and decoder an LSTM-based model, was carried out to get attribute specific trends and information. A Flask-based website allows the user to customize and select outfits according to their preference.
